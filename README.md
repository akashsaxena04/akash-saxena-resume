### LRU Cache Simulator

A system simulation project demonstrating the **Least Recently Used (LRU) cache replacement algorithm**, commonly used in operating systems and memory management.

Key Features:

* Simulates cache memory operations
* Implements the LRU replacement policy

* Visualizes how recently used data is prioritized in cache systems
